*Can you crack the password to get the flag?*

*Download the password checker here and you'll need the encrypted flag and the hash in the same directory too. Here's a dictionary with all possible passwords based on the password conventions we've seen so far.*

## Hints
*Opening a file in Python is crucial to using the provided dictionary.*  
Pythonでファイルを開くことは、提供されているディクショナリを利用する上で不可欠です。  

*You may need to trim the whitespace from the dictionary word before hashing. Look up the Python string function, **strip***  
ハッシュ処理を行う前に、辞書内の単語から余分な空白文字を削除する必要があるかもしれません。Pythonの文字列関連の関数を参照してください。 **strip**  

*The **str_xor** function does not need to be reverse engineered for this challenge.*  
このチャレンジにおいては、 **str_xor** 関数を逆コンパイルする必要はありません。  

## Answer  
☆Pythonのコード内で直接```dictionary.txt```を開く。  
```python
# 外部ファイルを読み込む（JavaのBufferedReaderみたいなもの）
# dictionary.txt を開いて、1行ずつループで回す
with open('dictionary.txt', 'r') as f:
    for line in f:
        # 1行読み込むと末尾に「改行文字(\n)」が入っているので、strip() で綺麗に削る
        pw = line.strip()
        
        # パスワードを自動チェック
        if level_5_pw_check_auto(pw):
            break # 正解が見つかったらループ終了！
```
### 1. ```with open(...) as f:```  
Javaの ```try-with-resources``` と同じです。ファイルを自動的にクローズ（```.close()```）してくれる安全な構文です。'```r```' は Read（読み込みモード）の意味です。  

### 2. ```for line in f:```  
Javaの ```while``` ((```line = br.readLine()) != null```) と全く同じ動きを、この短いfor文だけでやってくれます。ファイル内のパスワード候補を上から1行ずつ自動で取り出します。  

### 3. ```line.strip()``` 
Javaの ```line.trim()``` と同じです。テキストファイルから読み込んだ文字列の末尾にある目に見えない「改行コード（\n）」をきれいに消し去ってくれます。これをやらないと、ハッシュ値がズレて「incorrect」になってしまうので超重要です。  

ただし、```level_5_pw_check_auto()```はデフォルトでPythonファイルに含まれていた関数で、手入力で1つずつPWを入力するメソッドになっている。
